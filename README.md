# 🚀 Project Overview

Every successful machine learning project begins with clean, reliable data.

This project performs **Exploratory Data Analysis (EDA)** and **Feature Engineering** on the Google Play Store dataset to uncover hidden trends, identify data quality issues, and prepare the dataset for predictive analytics.

The notebook demonstrates practical data science skills that recruiters expect, including:

* Data Cleaning
* Missing Value Analysis
* Data Type Conversion
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Statistical Insights
* Business-Oriented Visualization

---

# 🎯 Problem Statement

The Google Play Store contains millions of applications across multiple categories.

The objective is to answer business questions such as:

* Which app categories are the most popular?
* Which apps receive the highest installs?
* Which categories earn the highest ratings?
* What data quality issues exist?
* How should raw data be prepared for machine learning?
